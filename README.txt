I. Cài các công cụ cần thiết cho dự án:
1. NodeJS & npm: https://nodejs.org/en/download/
- Phiên bản Node: 17.14.3
- Phiên bản npm: 6+
- Tools: Visual Studio Code

II. Giải nén file source-code 

III. Chạy api
1. Tại thư mục api, mở CMD or terminal
2. Gõ lệnh "npm start" để chạy chương trình

IV. Chạy client
1. npm install
2. Tại thư mục client, mở CMD or terminal
3. Gõ lệnh "npm start"

IV. Chạy admin
1. npm install
2. Tại thư mục admin, mở CMD or terminal
3. Gõ lệnh "npm start"
